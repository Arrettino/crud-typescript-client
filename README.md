# crud-typescript-client

## Requirements 📋

_You'll need node installed_

## Installation 🔧

_In the project directory, to install packages, you can run:_

```
$ npm install
```

## Start Application 📦

_In the main directory of the app, you can run:_

```
$ npm run start
```

_Then open: http://localhost:3000/_
